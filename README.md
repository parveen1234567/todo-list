📝 To-Do List Web Application

📌 Project Overview
This is a simple and interactive To-Do List Web App built using HTML, CSS, and JavaScript.
It allows users to add tasks, mark them as completed, remove tasks, and store them in localStorage so that the list persists even after refreshing the page.

🚀 Features
Add new tasks with a description.

Mark tasks as completed (with line-through effect).

Remove tasks from the list.

Tasks are saved in localStorage for persistence.

Responsive design for mobile and desktop.

Clean UI with hover effects and transitions.

⚙️ How It Works
Add Task

Type a description in the input box and click Add.

The task appears in the list with Completed and Remove buttons.

Mark as Completed

Clicking Completed toggles a line-through effect on the task text.

Remove Task

Clicking Remove deletes the task from the list.

LocalStorage Integration

Tasks are saved automatically in the browser’s localStorage.

On page reload, tasks are restored.
